Aviation Learning Management System (LMS)

A proprietary Java-based Learning Management System designed for delivering web-based TSA regulatory training to airport employees. This software has been used by over 100,000 airport employees in more than 60 FAR-139 airports around the United States including: Los Angeles International Airport (LAX), Dallas Fort Worth International Airport (DFW), Fort Lauderdale-Hollywood International Airport (FLL).

I was part of a team of developers on this project and over the course of a decade: designed, developed, and shipped numerous enhancements. Notable contributions as a developer and designer include:

  • A complete rewrite of the user interface HTML & CSS, including adding full mobile device support
  • Designed a B2C E-commerce interface
  • Designed a simple course compiler
  • Designed a location-based authorization system
  • Wrote developer REST API documentation

My Role

Frontend Developer, Designer, QA

Technology

Java, JavaScript, HTML5, CSS3, JSP, Bootstrap 4, Microsoft SQL Server, Apache Tomcat
