Enterprise Learning Management System (LMS)

A Java-based web-application backed by a Microsoft SQL database designed for delivering web-based training content to users. Core features include: user authentication, video-based training delivery, reporting, email notifications, and a direct-to-customer e-commerce module. I was part of a team of developers on this project and over the course of a decade designed, developed, and tested numerous enhancements. Notable contributions include:

  • A complete rewrite of the user interface — transitioning from server-side rendering via JSP to a single page application using RESTful APIs and the Vue.js framework
  • An e-commerce module which connects to Stripe and Authorize.net payment gateways
  • A new suite of RESTful APIs
  • A graphical reporting dashboard

This software has been used in the insurance, mining, higher education, and healthcare industries.
