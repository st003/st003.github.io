Restaurant Ordering System (CS50W)

A Python Django-based web-application for making online orders for a pizza restaurant. This was one of the assignments for HarvardX CS50W. The core goals were to create a web-application with customer registration, a menu, and a means of placing an order. Additionally, restaurant staff also needed to track which orders were paid and use the Django admin to make changes to the menu.

My Role

I designed and developed this application based on a course rubrik

Technology

Python, JavaScript, Django, Bootstrap, SQLite
