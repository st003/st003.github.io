Admin Tool #2

This is simple Flask-based web-application that listens to HTTP POSTs on a designated endpoint then prints the incomming data to the GUI via websockets. Tis primary use-case was for testing webhooks for other internal applications.

I designed, wrote, deployed, and maintained the entirety of this application. Technology: Python, JavaScript, Flask, socket.io, Gunicorn, Nginx, CentOS. This project is not open source.
