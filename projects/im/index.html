Instant Messenger (CS50w)

A Python Flask-based web-application for instant messenging. This was one of the assignments for HarvardX CS50W. The application supports self-registration, multiple users, simultaneously instant messenging, group chats, and direct messages. User can create their own group chat channels as well. One of the core technologies for this project was the use of websockets to create push notifications so a user did not need to refresh the page to see new messages.
