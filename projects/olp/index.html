Stuart Calcote | Full Stack Web Developer

Learning Management System (LMS)

A Java-based web-application backed by a Microsoft SQL database designed for delivering web-based training content to users. Core features include: user authentication, video-based training delivery, reporting, email notifications, and a direct-to-customer e-commerce module. I was part of a team of developers on this project and over the course of a decade, designed and tested numerous enhancements (including the e-commerce module), project managed a small development team, and eventually re-wrote the entire user interface, transitioning from server-side rendering using JSP to a single page application using the Vue.js framework. In addition I've contributed to multiple RESTful web-service API endpoints.

This LMS has been used in the insurance, mining, higher education, and healthcare industries.
